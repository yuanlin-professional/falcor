# MitsubaImporter

这是一个用于 Mitsuba 3 XML 场景文件的场景导入器。
Falcor 仅支持 Mitsuba 3 中实现的非常有限的功能子集，
因此场景转换远非完美。下面的列表概述了此导入器当前支持的对象和参数。
