# Vector 源码文档

> 路径: `Source/Falcor/Utils/Math/Vector.h`
> 类型: C++ 头文件
> 模块: Utils/Math

## 功能概述

统一的向量头文件，聚合了向量类型定义和向量运算。作为其他模块引用向量功能的入口。

## 类与接口

本文件为聚合头文件，仅包含以下两个头文件：

- `VectorTypes.h` - 向量类型定义
- `VectorMath.h` - 向量运算函数

## 依赖关系
### 本文件引用
- `VectorTypes.h`
- `VectorMath.h`

### 被以下文件引用
- 几乎所有 Falcor 模块都依赖此文件
