# Toy.ps.slang 着色器文档

> 路径: `Source/Samples/ShaderToy/Toy.ps.slang`
> 类型: Slang 像素着色器
> 模块: Samples/ShaderToy

## 功能概述

ShaderToy 示例的程序化像素着色器,生成一个模拟红细胞(RBC)运动效果的动画。着色器使用基于哈希的伪随机函数生成圆形粒子网格,每个粒子带有抖动运动,并使用自定义的多项式颜色映射函数模拟红细胞的深红到浅红色变化。

## 结构体与接口

### 常量缓冲区 `ToyCB`

| 成员 | 类型 | 说明 |
|------|------|------|
| `iResolution` | `float2` | 渲染目标分辨率 |
| `iGlobalTime` | `float` | 全局时间(秒) |

## 函数

| 函数签名 | 说明 |
|----------|------|
| `float rand(float n)` | 一维伪随机函数,基于正弦哈希 |
| `float2 rand2(float2 p)` | 二维伪随机函数,返回二维随机向量 |
| `float4 getRBCColor(float x)` | 红细胞颜色映射,使用六次多项式从深红到浅红插值 |
| `float4 mainImage(float2 fragCoord)` | 主渲染函数,计算每个像素的红细胞效果 |
| `float4 main(float2 texC: TEXCOORD) : SV_TARGET` | 像素着色器入口,将纹理坐标转换为像素坐标后调用 `mainImage` |

## 预定义宏

| 宏名 | 值 | 说明 |
|------|------|------|
| `v4DarkRed` | `float4(0.2, 0.1, 0.1, 1.0)` | 深红色 |
| `v4LightRed` | `float4(0.7, 0.1, 0.1, 1.0)` | 浅红色 |

## 依赖关系 / import

无外部依赖。

## 实现细节

- 使用频率为 7.0 的网格将屏幕空间划分为单元格,每个单元格中心放置一个圆形粒子。
- 粒子半径通过伪随机函数在基础值上添加变化,避免视觉上的规律感。
- 粒子位置通过多层正弦波叠加实现平滑的抖动运动,模拟液体中的布朗运动。
- `getRBCColor` 使用六次多项式拟合红细胞的径向颜色变化,中心偏暗,边缘偏亮。
- 使用 `smoothstep` 在粒子边界处实现平滑过渡,避免硬边缘。
- 黑白闪烁效果 (`black_or_white`) 基于随机值和时间的正弦组合(在当前代码中计算但未直接应用到最终输出)。
